Ingeniería Web I Unidad 3 — CSS Fundamentos

Ejercicio + Tarea

Estilizando un artículo de blog con CSS

Los estudiantes reciben el HTML de un artículo ya estructurado y deben aplicar CSS.

⏱ Ejercicio: ~45 min

Contexto de la actividad

¿Qué aprenderán?

En esta actividad, el HTML ya está listo: el estudiante debe leer, comprender y aplicar estilos a un documento existente sin modificar su estructura.

💡
El archivo blog-articulo.html contiene el artículo completo. No se permite modificar el HTML, excepto para enlazar el archivo estilos.css dentro del <head>.
Estructura de archivos esperada
semana-05/
  ├── blog-articulo.html  ← proporcionado
  └── estilos.css  ← el estudiante lo crea

Traducir los requerimientos visuales a propiedades CSS.

  1. Crear y enlazar el archivo CSS

    En la misma carpeta donde tienes el HTML, crea un archivo llamado estilos.css. Luego, ve al <head> de tu archivo HTML y descomenta la etiqueta <link> para asociar tu hoja de estilos.

  2. Importar fuentes y configurar el body

    Copia el siguiente código en la primera línea de tu estilos.css para importar las tipografías necesarias:

    @import url('https://fonts.googleapis.com/css2?family=Lato:wght@400;700&family=Playfair+Display:wght@700&display=swap');

    Selecciona la etiqueta body. Asígnale la fuente principal 'Lato' (con sans-serif como respaldo). Define el tamaño de fuente a 16px y la altura de línea a 1.7. Aplica el color "texto" de tu paleta al texto, y el color "fondo" al background. Por último, quítale todo el margen exterior por defecto (0).

  3. Estilizar el encabezado del sitio

    Selecciona el contenedor con el id site-header. Asígnale el color "header" de tu paleta como fondo. Dale un relleno (padding) de 16px arriba y abajo, y 32px a los lados. Usa Flexbox para alinear sus elementos hijos: distribuye el espacio máximo entre ellos y céntralos verticalmente.

    Para el logo (id site-logo): quítale el subrayado, usa el color "texto claro", dale un tamaño de 1.4rem y ponlo en negrita.

    Para el menú de navegación (la lista ul dentro del nav del header): quítale las viñetas por defecto, quita los márgenes y rellenos. Conviértelo en un contenedor flex para que los ítems se coloquen en fila y dale un espacio (gap) de 24px entre ellos.

    A los enlaces dentro de esa lista: dales el color "acento suave", quítales el subrayado y pon el tamaño de fuente en 0.95rem.

  4. Centrar y limitar el ancho del contenido principal

    Selecciona la etiqueta main. Limita su ancho máximo a 760px. Céntralo en la pantalla usando márgenes automáticos a los lados (y 0 arriba/abajo). Finalmente, ponle un relleno interno de 40px arriba/abajo y 24px a los lados.

  5. Jerarquía tipográfica: Títulos

    Agrupa los selectores h1, h2 y h3 para aplicar estilos compartidos: usa la fuente 'Playfair Display' (con serif de respaldo), aplícales el color "header" de tu paleta y establece una altura de línea de 1.25.

    Luego, dales estilos específicos a cada nivel:

    • Al h1: tamaño de 2.2rem y margen inferior de 16px.
    • Al h2: tamaño de 1.5rem, margen superior de 40px y margen inferior de 12px.
    • Al h3: tamaño de 1.15rem, margen superior de 28px y margen inferior de 8px.
  6. Tratamiento de la imagen de portada

    A la clase .article-cover, dale un margen exterior de 32px arriba y abajo (y 0 a los lados).

    Selecciona la imagen que está dentro de .article-cover: haz que ocupe el 100% de su contenedor de ancho, que la altura se calcule automáticamente y aplícale un radio de borde de 8px.

    Para la descripción de la imagen (la etiqueta figcaption): dale un tamaño de 0.85rem, usa el color "texto atenuado", alinea el texto al centro, añádele un margen superior de 8px y cambia el estilo de la fuente a cursiva.

  7. Destacar la cita (Blockquote)

    A la etiqueta blockquote ponle el color "acento suave" de fondo y aplícale un borde izquierdo sólido de 4px usando tu color "acento". Dale un relleno de 20px arriba/abajo y 24px a los lados. Sepáralo del resto del contenido con un margen de 32px arriba y abajo. Redondea sus bordes a 8px, pero ojo: solo los bordes derechos (superior e inferior).

    Al párrafo que está dentro del blockquote: ponlo en cursiva, tamaño 1.1rem y un margen inferior de 8px.

    A la referencia de la cita (la etiqueta cite): tamaño de 0.85rem, aplícale el color "acento", fórza el estilo de fuente a normal (para quitarle la cursiva heredada) y ponlo en negrita.

  8. Formato del código (Pre y Code)

    Para el bloque pre: usa un color de fondo oscuro (como #1e1e1e) y un color de texto claro (como #d4d4d4). Dale un relleno de 24px en todas direcciones y un redondeo de bordes de 8px. Muy importante: asegúrate de manejar el desbordamiento añadiendo scroll horizontal automático. Usa una familia tipográfica monoespaciada ('Courier New', monospace), tamaño 0.88rem, altura de línea 1.7 y un margen exterior de 24px arriba/abajo.

    Para la etiqueta code que está en línea (dentro de los párrafos): usa también la fuente monoespaciada, tamaño de 0.88em. Ponle de fondo el color "acento suave" y texto del color "acento". Dale un relleno de 2px arriba/abajo y 6px a los lados, con un redondeo de 4px.

    Cuidado con la herencia: El code que está dentro del pre no debe verse como el inline. Selecciona específicamente el code dentro del pre y resetea su fondo (transparente), su color (heredado), su tamaño (heredado) y quítale el relleno (0).

  9. Listas del artículo y Etiquetas (Tags)

    A las listas (ul y ol) que estén directamente dentro del contenedor con clase .article-body: dales un relleno izquierdo de 24px y un margen exterior de 16px arriba y abajo. Al li que está dentro de ellas, ponle un margen inferior de 8px.

    A la lista con clase .tags: quítale las viñetas, pon su relleno en cero y dale un margen de 16px 0 0. Usa Flexbox para que los elementos fluyan en fila pero permitan saltos de línea, con un espacio (gap) de 8px.

    A la clase .tag (cada píldora individual): haz que su comportamiento de display sea inline-block. Usa el color "acento suave" de fondo y el color "acento" para el texto. Añádele un borde sólido de 1px (puedes usar un color derivado u oscurecer un poco el acento suave). Dale un relleno de 4px arriba/abajo y 14px a los lados. Redondea totalmente sus bordes (usa un valor grande como 100px), define el texto en 0.82rem y con peso de 600.

  10. El pie de página (Footer)

    Selecciona el id site-footer. Aplícale el mismo color "header" de fondo que usaste arriba y utiliza el color "acento suave" (o "texto claro") para el texto. Centra el texto, ponle un relleno interno de 24px en todos los lados, ajusta el tamaño a 0.85rem y sepáralo del artículo dándole un margen superior de 64px.
